## TROUBLESHOOTING

**"npm not found"**  
→ Install Node.js: https://nodejs.org/

**"Port 3000 in use"**  
→ Run: `npx kill-port 3000`

**Build errors**  
→ Delete `node_modules` and `.next`, then `npm install`
